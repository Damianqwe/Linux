# Linux
## list of commands

`ls` - list dir
`chmod` - change modification
`useradd` - add user
`groupadd` - add group
`usermod` - user modify
`su` - switch user
`passwd` - set password
`touch` - update last time file 
`pwd` - print wokring dir
`cd` - change dir
`cat` - show content of file
`mkdir` - make dir
`chown` - change membership
`env` - show veriables
`rm` - remove
`cp` - copy
`mv` - move
